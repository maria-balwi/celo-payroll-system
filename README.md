# CELO Payroll System

## Introduction
**CELO Payroll System with Attendance Monitoring** is a comprehensive solution designed to streamline workforce management and payroll processing.  
This system integrates real-time attendance tracking with automated payroll calculations to ensure accuracy and efficiency in managing employee work hours and compensation.

With features such as biometric attendance capture, leave management, and customizable payroll settings, it minimizes errors, reduces administrative workload, and provides valuable insights into workforce productivity.  
Tailored for **CELO Business Solutions Inc.**, this system enhances overall operational efficiency and employee satisfaction.

---

## Features
- Employee and Administrative side access  
- Add, Update, Remove, and View Employees  
- Manage Attendance and Overtime  
- Manage Leave Applications  
- Manage Change Shift Requests  
- Generate Payroll  
- Generate Pay Slips  
- Generate Reports  
- User Management System

---

## Technologies Used
- **Frontend:** HTML, CSS (Tailwind CSS), JavaScript, jQuery  
- **Backend:** PHP  
- **Database:** MySQL  
- **Libraries/Plugins:** Bootstrap Select, DataTables
